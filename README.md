# Hörnchen-Tracking

This repository contains code, data, and documentation for the **Squirrel Tracking** study project at the University of Münster. The goal of this project is to detect and track squirrels in video footage recorded at nest boxes. We explore and compare multiple computer vision approaches — including background subtraction, frame differencing, and YOLO-based object detection — to analyse squirrel behaviour such as entry timing, presence duration, and interaction with objects inside the box.
